This course will give an introduction to computer game development starting from concept development to implementation of a playable game prototype. Both the aesthetic and
technical aspects of game development will be covered. The aesthetic component of the course will focus on story and character development, game mechanics, game play and
interface design and content creation for games. The technical component will focus on programming tools and concepts for games.

Students will start from basic game idea and at the end of the program develop their own playable prototype.
